Project Description:
The website ARTISAN INTERIORS is a functional Furniture and  Interior Design website developed as our final project.

User Interface:
-A dark theme with gold accents used.
-Smooth navigation between home,collection,contact page instantly using JS.

Functionality:
-Add items to the cart.
-Calculation of totals.
-Remove items.
-A functional contact page with validation.
-User login form

Pages:
-Home page with hero section.
-Collection page in which items are displaying by grid.
-Lookbook page included as a gallery.
-About page shows our services and history.
-Contact page included for customers contact.

Technologies used:
-HTML
-CSS
-JavaScript
-FontAwesome library for icons.
